# Krish Handicraft Store

A static website project showcasing traditional Indian handicraft products.

## 🌟 Live Website
🔗 https://ishwarya02-student.github.io/Krish-Handicraft-Store/

---

## 📌 Project Overview

This project represents an online handicraft store featuring:

- Traditional textiles
- Hand-painted pottery
- Rajasthani jewellery
- Wooden carvings
- Decorative items

The website is built as a static frontend project and hosted using GitHub Pages.

---

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript
- GitHub Pages (for deployment)
